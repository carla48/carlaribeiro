# carlaribeiro
Para rodar a aplicação: git, node e npm instalado, banco de dados mysql local.
Aqui eu tenho um mysql: localhost:3306/costdb

Se você tiver um mysql local, você pode criar o costdb. As tabelas e sequências serão criadas ao executar a aplicação.

Baixar:
git clone https://github.com/carla48/carlaribeiro.git

Observe que dois diretórios serão criados:
cost e costweb

Ir no diretório cost/target e executar (para inicializar o servidor):
java -jar gasto-0.0.1-SNAPSHOT.jar

Ir no diretório costweb e executar:
npm install
ng serve

Para abrir a página web, http://localhost:4200/
